# 💊 안심복약 AI (Medi-Check)
> **"복잡한 약 관리, 사진 한 장으로 끝내는 AI 기반 개인 맞춤형 복약 가이드"**

## 📝 프로젝트 소개 (Project Introduction)
고령화 사회로 접어들며 만성질환자의 복용 약물 개수가 늘어나고, 영양제 섭취 또한 보편화되었습니다. 하지만 약물 간의 상호작용(궁합)을 개인이 파악하기는 매우 어렵습니다. 

**'안심복약 AI'**는 카메라 스캔 한 번으로 복잡한 약 이름을 자동 등록하고, 국가 공인 데이터를 바탕으로 약물 간 충돌 위험을 실시간 분석하여 안전한 복약 환경을 제공하는 헬스케어 솔루션입니다.

---

## ✨ 핵심 기능 (Key Features)

### 📸 약 봉투/처방전 OCR 자동 등록
* **기능:** 사용자가 약 봉투나 처방전을 촬영하면 AI가 의약품 명칭만 추출하여 리스트업합니다.
* **기술:** **Google Cloud Vision API**로 텍스트를 추출하고, **GPT-4o**를 통해 비정형 데이터에서 유효한 약물 이름을 정제합니다.
* **가치:** 수동 입력의 번거로움을 제거하여 디지털 취약계층의 진입장벽을 낮춥니다.

### ⚠️ 약물 상호작용(DUR) 실시간 분석
* **기능:** 등록된 약물 간의 병용 금기, 성분 중복 여부를 체크하여 위험을 경고합니다.
* **기술:** **식약처 공공데이터(DUR API)** 및 약학정보원 데이터를 실시간으로 조회하여 매칭합니다.
* **가치:** "A약과 B약을 함께 드시면 위험합니다"와 같은 구체적인 가이드를 통해 약물 사고를 예방합니다.

### 🗓️ 지능형 복용 타임라인 & 스마트 알림
* **기능:** 약물별 복용 지침(식전/식후 등)을 분석해 통합 스케줄을 생성하고 알림을 보냅니다.
* **기술:** 사용자의 식사 시간 설정과 약물 특성을 결합한 **스케줄링 알고리즘**을 적용합니다.
* **가치:** 단순 알람을 넘어 맥락 기반의 푸시 알림으로 복약 순응도를 극대화합니다.

---

## 🏗️ 시스템 아키텍처 (System Architecture)



1. **Client**: React Native (Expo) 기반 크로스 플랫폼 앱
2. **Server**: FastAPI를 통한 고성능 비동기 처리 및 API 중계
3. **External Engines**: 
    * **OCR**: Google Cloud Vision API
    * **Reasoning**: OpenAI GPT-4o API
    * **Medical Data**: 공공데이터포털(DUR API), 약학정보원 DB 연동

---

## 🛠 기술 스택 (Tech Stack)

### **Frontend**
* **React Native (Expo)**: 카메라 API 제어 및 Expo Notifications를 활용한 실시간 알림 구현.

### **Backend**
* **FastAPI (Python)**: 공공 데이터 API 호출 및 AI 모델 파이프라인 최적화.

### **Database**
* **PostgreSQL**: 약물 성분 간의 관계 및 사용자별 맞춤형 복용 데이터 관리.

### **AI & Data Processing**
* **Google Cloud Vision API**: 사진 속 텍스트 추출 (OCR 핵심 엔진).
* **OpenAI API (GPT-4o)**: 추출된 텍스트 정제 및 사용자 친화적인 복용 주의사항 요약 제공.
* **공공데이터포털 (DUR API)**: 식약처 제공 병용 금기 및 임부 금기 정보 실시간 연동.

---

## 💡 기대 효과 (Expected Effects)
* **약물 사고 예방:** 병용 금기 약물 섭취로 인한 부작용을 사전에 차단하여 시니어 및 만성질환자의 안전 보장.
* **복약 순응도 향상:** 파편화된 정보를 하나의 타임라인으로 관리하여 약을 거르는 횟수 최소화.
* **디지털 헬스케어 경험:** 어려운 전문 정보를 AI를 통해 일반인이 이해하기 쉬운 언어로 전달.
