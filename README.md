# 3-1 프로젝트 주제: 잊혀진 구독 찾기 

## 프로젝트 소개
"본인도 모르게 새어나가는 돈, 이제 AI가 찾아드립니다."

OTT, 음악, 클라우드 등 현대인들이 이용하는 수많은 구독 서비스. 하지만 정작 잘 쓰지 않으면서 매달 결제되는 '방치된 구독'들이 많습니다. **'잊혀진 구독 찾기'**는 흩어진 소액 정기 결제 내역을 시각화하고, AI 분석을 통해 불필요한 지출을 최적화해 주는 통합 구독 관리 모바일 앱입니다.

## 핵심 기능

* **🤖 AI 기반 비용 최적화 레포트**
  * 사용자가 입력한 데이터나 영수증 텍스트를 AI가 분석하여 카테고리(OTT, 음악 등)를 자동 분류합니다.
  * "비슷한 서비스(예: 넷플릭스 vs 디즈니+)를 중복 이용 중입니다. 하나를 해지하면 월 15,000원을 아낄 수 있어요!"와 같은 맞춤형 비용 최적화 인사이트를 제공합니다.
* **⏰ 결제 전 '생존 신고' (Active Alert)**
  * 구독료가 빠져나가기 2~3일 전, 스마트폰 푸시 알림을 통해 사용자에게 "이번 달에도 계속 구독하시겠어요?"라고 질문하여 해지 골든타임을 확보해 줍니다.
* **🩺 '방치된 구독' 상태 체크 (Health Check)**
  * 한 달에 한 번, 등록된 구독 리스트를 점검하는 푸시 알림을 발송합니다.
  * 사용자가 "지난 한 달간 한 번도 안 썼음"을 선택하면 해당 서비스 옆에 `[⚠️ 해지 권장]` 뱃지를 부여하여 관리를 유도합니다.

## 기술 스택

### Frontend 
* **React Native (Expo):** iOS 및 Android 크로스 플랫폼 앱 개발
  * 💡 **Expo란?:** React Native로 앱을 만들 때 필요한 복잡한 초기 환경 세팅(Android Studio, Xcode 등)을 모두 생략하게 해주는 강력한 개발 도구입니다. 기존에 알던 리액트(React) 문법을 그대로 사용할 수 있으며, 'Expo Go' 앱을 통해 내 스마트폰에서 실시간으로 테스트하며 개발할 수 있어 진입 장벽이 낮고 개발 속도가 매우 빠릅니다.

* **Expo Notifications:** 서버 스케줄러 없이 기기 내부에서 작동하는 로컬 푸시 알림 구현

### Backend & Database
* **Firebase (Firestore):** NoSQL 기반 사용자별 구독 리스트 및 결제 데이터 저장
* **Firebase Authentication:** 간편한 사용자 로그인(OAuth) 처리

### AI & Data Processing
* **OpenAI API (`gpt-4o-mini`):** `JSON` 모드를 활용한 영수증 텍스트 파싱(서비스명, 결제액, 카테고리 추출) 및 중복 구독 분석 리포트 생성

* "결제 문자 복붙" + AI 분석
=> 사용자가 카드사 결제 문자나 이메일 영수증 텍스트를 복사해서 우리 앱에 '붙여넣기' 합니다.

작동 원리: 복붙한 텍스트를 백엔드(ChatGPT API)로 보내고 AI가 텍스트를 분석해서 "아! 넷플릭스에서 17,000원이 오늘 결제됐네!" 하고 알아서 데이터를 뽑아내 Firebase DB에 저장해 줍니다.

장점: 아이폰(iOS), 안드로이드 모두에서 완벽하게 작동하고 구현이 아주 쉽습니다. 핵심 기능인 'AI 분석'을 어필하기에도 가장 good

## 💡 기대 효과
* 파편화된 구독 서비스의 직관적인 통합 관리
* AI의 능동적인 개입을 통한 실질적인 고정 지출 절감
* 불필요한 결제를 사전에 방지하는 긍정적인 사용자 경험(UX) 제공
