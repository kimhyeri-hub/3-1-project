# 💊 안심복약 AI (Medi-Check)
> **"복잡한 약 관리, 사진 한 장으로 끝내는 AI 기반 개인 맞춤형 복약 가이드"**

## 📝 프로젝트 소개 (Project Introduction)
고령화 사회로 접어들며 만성질환자의 복용 약물 개수가 늘어나고, 영양제 섭취 또한 보편화되었습니다. 하지만 약물 간의 상호작용(궁합)을 개인이 파악하기는 매우 어렵습니다. 

**'안심복약 AI'**는 카메라 스캔 한 번으로 복잡한 약 이름을 자동 등록하고, 국가 공인 데이터를 바탕으로 약물 간 충돌 위험을 실시간 분석하여 안전한 복약 환경을 제공하는 헬스케어 솔루션입니다.

---

## ✨ 핵심 기능 (Key Features)

### 📸 지능형 약물 스캔 & 정보 자동 출력 (Smart Scanning)
* **기능:** 앱 내 카메라 권한을 허용하여 약 봉투나 처방전을 스캔하면, 실시간으로 해당 약물의 상세 정보(성분, 효능, 주의사항)를 화면에 즉시 출력합니다.
* **기술:** **Google Cloud Vision API**를 통해 이미지 내 텍스트를 검출하고, **GPT-4o**가 문맥을 분석해 실제 의약품 명칭만 추출합니다.
* **가치:** 사용자가 직접 약 이름을 검색할 필요 없이, 시각적 피드백을 통해 즉각적인 약물 정보 확인이 가능합니다.

### ⚠️ 약물 상호작용(DUR) 실시간 분석
* **기능:** 새로 스캔한 약이 기존 복용 중인 약물과 함께 먹었을 때 위험(병용 금기, 성분 중복 등)이 있는지 체크하여 즉시 경고합니다.
* **기술:** **식약처 공공데이터(DUR API)** 및 약학정보원 데이터를 실시간으로 조회하여 매칭 알고리즘을 수행합니다.
* **가치:** 상충하는 약물 섭취로 인한 부작용을 사전에 차단하여 시니어 및 만성질환자의 안전을 보장합니다.

### 🗓️ 지능형 복용 타임라인 & 스마트 알림
* **기능:** 스캔된 약들의 복용 지침(식전/식후 등)을 분석해 최적화된 통합 복약 스케줄을 자동 생성합니다.
* **기술:** 사용자 생활 패턴(식사 시간 등)과 약물 특성을 결합한 **스케줄링 알고리즘** 기반 푸시 알림을 발송합니다.
* **가치:** 파편화된 복용 정보를 하나의 타임라인으로 통합하여 약을 거르는 횟수를 최소화합니다.

---

## 🏗️ 시스템 아키텍처 (System Architecture)

1. **Client**: React Native 기반 카메라 인터페이스 및 실시간 정보 렌더링
2. **Server**: 비동기 처리를 통한 OCR 데이터 파싱 및 외부 API 중계 로직
3. **Data**: 식약처 공공데이터(DUR) 및 GPT-4o 기반 지식 정제

---

## 🛠 기술 스택 (Tech Stack)

### **Frontend**
* **React Native (Expo)**: Camera API 제어 및 실시간 스캔 UI 구현
* **Expo Notifications**: 복약 주기 관리 및 스마트 알람 시스템

### **Backend & Database**
* **FastAPI (Python)**: 고성능 비동기 API 서버 및 AI 파이프라인 최적화
* **PostgreSQL**: 약물 성분 관계 및 사용자 복용 로그 관리

### **AI & Data Processing**
* **Google Cloud Vision API**: OCR(광학 문자 인식) 엔진
* **OpenAI API (GPT-4o)**: 의약품 명칭 정제 및 사용자 맞춤형 가이드 생성
* **공공데이터포털 (DUR API)**: 국가 공인 의약품 안전성 정보 연동

## 💡 기대 효과 (Expected Effects)
* **정확한 정보 전달:** 어려운 의약 정보를 AI를 통해 일반인이 이해하기 쉬운 언어로 자동 변환하여 제공합니다.
* **복약 순응도 향상:** 스캔만으로 등록되는 편리한 UX를 통해 사용자의 자발적인 복약 관리를 유도합니다.
* **안전한 투약 환경:** 실시간 성분 대조를 통해 오투약 및 약물 충돌 사고를 획기적으로 방지합니다.
